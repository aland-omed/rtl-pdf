# Code of Conduct

We pledge to provide a welcoming, harassment-free project for everyone,
regardless of identity, language, writing system, location, background, or
experience.

Be respectful, discuss typography and language differences with evidence, and
welcome corrections from native readers. Do not treat one language, dialect,
script, or direction as superior.

Harassment, discriminatory language, personal attacks, deliberate disruption,
and publishing private information are unacceptable. Maintainers may edit or
remove contributions and restrict participation when necessary. Report conduct
concerns privately through the repository owner's GitHub profile.

This policy is adapted from Contributor Covenant 2.1.
