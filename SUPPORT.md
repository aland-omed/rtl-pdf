# Support

Use [GitHub Discussions](https://github.com/aland-omed/rtl-pdf/discussions) for
integration questions, font recommendations, and ideas that are not yet defined
bugs or features.

Use the issue templates for reproducible rendering bugs and scoped feature
requests. Include a minimal example and remove all private document data.

Report security vulnerabilities privately as described in
[SECURITY.md](SECURITY.md).

This is a community-maintained project. Response times are best effort, and
clear reproductions receive priority.
